Gas Mask Help


The Hosts file is a text file used by an operating system for mapping hostnames to IP addresses.

Gas Mask is an Hosts files manager.

It can modify the system Hosts file, edit various Hosts files, manage several sets, use hosts files downloaded from Internet.
You can also quickly switch to set one of them as the current Host file.


For more information see http://en.wikipedia.org/wiki/Hosts_file
